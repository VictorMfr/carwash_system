Bienvenido a la documentación oficial del sistema LA MANO DE DIOS.

¿Qué encontrarás aquí?​

Esta documentación te guiará a través de todos los módulos y funcionalidades del sistema, desde la configuración inicial hasta el uso avanzado de cada componente.

Características principales​

  • Gestión de Inventario: Control completo del stock y productos
  • Módulo Financiero: Seguimiento de cuentas y transacciones
  • Gestión de Servicios: Administración de servicios y mantenimientos
  • Control de Usuarios: Sistema de roles y permisos
  • Dashboard Integrado: Vista general del sistema

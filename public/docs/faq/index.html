Preguntas Generales​

¿Qué es el Sistema de Gestión Empresarial?​

El Sistema de Gestión Empresarial es una aplicación web desarrollada con Next.js que te permite gestionar diferentes aspectos de tu empresa de manera integral, incluyendo inventario, finanzas, servicios y usuarios.

¿Qué navegadores son compatibles?​

El sistema es compatible con:

  • Chrome 90+
  • Firefox 88+
  • Safari 14+
  • Edge 90+

¿Necesito instalar software adicional?​

No, el sistema es una aplicación web que funciona directamente en tu navegador. Solo necesitas una conexión a internet.

Preguntas sobre Instalación​

¿Cómo instalo el sistema?​

  1. Clona el repositorio
  2. Instala las dependencias con npm install
  3. Configura las variables de entorno
  4. Ejecuta las migraciones de base de datos
  5. Inicia el servidor con npm run dev

¿Qué base de datos necesito?​

El sistema soporta:

  • PostgreSQL (recomendado)
  • MySQL
  • SQLite (para desarrollo)

¿Cómo configuro la base de datos?​

Crea un archivo .env.local con:

DATABASE_URL="postgresql://usuario:password@localhost:5432/sistema_gestion"

Preguntas sobre Uso​

¿Cómo creo un nuevo usuario?​

  1. Ve a Usuarios en el menú lateral
  2. Haz clic en "Nuevo Usuario"
  3. Completa el formulario
  4. Asigna un rol apropiado
  5. Guarda el usuario

¿Cómo cambio mi contraseña?​

  1. Haz clic en tu avatar
  2. Selecciona "Mi Perfil"
  3. Ve a la sección "Seguridad"
  4. Ingresa tu contraseña actual
  5. Ingresa tu nueva contraseña
  6. Confirma el cambio

¿Cómo agrego un nuevo producto?​

  1. Ve a Stock en el menú lateral
  2. Haz clic en "Nuevo Producto"
  3. Completa la información del producto
  4. Asigna una categoría
  5. Guarda el producto

Preguntas sobre Reportes​

¿Cómo genero un reporte?​

  1. Ve a Reportes en el menú lateral
  2. Selecciona el tipo de reporte
  3. Configura los filtros
  4. Genera el reporte
  5. Exporta en el formato deseado

¿Qué formatos de exportación están disponibles?​

  • PDF: Para imprimir o compartir
  • Excel: Para análisis detallado
  • CSV: Para importar en otros sistemas

¿Cómo programo reportes automáticos?​

  1. Ve a Configuración > Reportes
  2. Selecciona el tipo de reporte
  3. Configura la frecuencia
  4. Establece los destinatarios
  5. Activa la programación

Preguntas sobre Seguridad​

¿Cómo protejo mi información?​

  • Usa contraseñas seguras
  • Habilita HTTPS en producción
  • Configura firewall apropiadamente
  • Mantén el sistema actualizado

¿Cómo configuro la autenticación?​

  1. Ve a Configuración > Seguridad
  2. Configura las políticas de contraseñas
  3. Establece la duración de sesiones
  4. Configura la autenticación de dos factores

¿Cómo audito las actividades?​

  1. Ve a Usuarios > Auditoría
  2. Revisa las actividades de usuarios
  3. Filtra por usuario, fecha o tipo
  4. Exporta los logs si es necesario

Preguntas sobre Integraciones​

¿Cómo integro con otros sistemas?​

  1. Ve a Integraciones en el menú lateral
  2. Selecciona el tipo de integración
  3. Configura los parámetros
  4. Prueba la conexión
  5. Activa la integración

¿Cómo uso la API?​

  1. Obtén un token de acceso
  2. Usa los endpoints disponibles
  3. Incluye el token en las requests
  4. Maneja los errores apropiadamente

¿Cómo configuro webhooks?​

  1. Ve a Integraciones > Webhooks
  2. Crea un nuevo webhook
  3. Configura la URL de destino
  4. Selecciona los eventos
  5. Activa el webhook

Preguntas sobre Problemas​

¿Qué hago si no puedo iniciar sesión?​

  1. Verifica tu usuario y contraseña
  2. Asegúrate de que tu cuenta esté activa
  3. Contacta al administrador
  4. Revisa la configuración de la base de datos

¿Qué hago si el sistema está lento?​

  1. Revisa la configuración de la base de datos
  2. Verifica el uso de memoria
  3. Optimiza las consultas
  4. Considera usar caché

¿Qué hago si pierdo datos?​

  1. Revisa los respaldos automáticos
  2. Restaura desde el último respaldo
  3. Contacta al soporte técnico
  4. Revisa los logs del sistema

Preguntas sobre Soporte​

¿Cómo contacto al soporte?​

  • Email: soporte@empresa.com
  • Teléfono: +1-800-123-4567
  • Chat: Disponible en la aplicación
  • GitHub: Reporta problemas en el repositorio

¿Cómo reporto un problema?​

  1. Ve a Soporte > Reportar Problema
  2. Describe el problema detalladamente
  3. Incluye capturas de pantalla
  4. Especifica los pasos para reproducir
  5. Envía el reporte

¿Cómo solicito una nueva funcionalidad?​

  1. Ve a Soporte > Solicitar Funcionalidad
  2. Describe la funcionalidad deseada
  3. Explica el caso de uso
  4. Incluye ejemplos si es posible
  5. Envía la solicitud
